# cifar10-CNN with TensorFlow
Simple CNN using Tensorflow on the cifar10 dataset
the dataset has 60000 labeled images and their classes are
    0: "airplane",
    1: "automobile",
    2: "bird",
    3: "cat",
    4: "deer",
    5: "dog",
    6: "frog",
    7: "horse",
    8: "ship",
    9: "truck",
